HTML Forms Tutorial

2. Form HTML Elements & Attributes

  • The main form element <form action="action.page.php" method="POST">
  • Full Name input <label for="fname">First name:</label> <input type="text" id="fname" name="fname" placeholder="Jacey Isaac" required>
  • Password field <label for="pass">Password:</label> <input type="password" id="pass" name="pass" maxlength="12">
  • Email input <label for="email">Email:</label> <input type="email" id="email" name="email" placeholder="example@email.com">
  • Country selection <select id="country" name="country"> <option value="php">Philippines</option> <option value="usa">United States</option> </select>
  • Message textarea <textarea id="message" name="message" rows="4"></textarea>
