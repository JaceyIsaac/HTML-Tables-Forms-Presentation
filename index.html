HTML Table Tutorial

1. Table HTML Elements & Attributes

  • Heading element to display "Weekly Date Adventure Planner" <h1>
  • The main table element <table>

    Attributes used:

    style="background-color:#b05f54;" align="center" width="700"
  • Defines a row in the table <tr>
  • Table header cells for each day of the week <th>

    Attributes used:

    width="300" (sets column width)
  • Table data cells containing date plans <td>

    Attributes used:

    align="center" (centers the text within the cell)
